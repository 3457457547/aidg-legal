English

This game runs on the user's local PC. The game itself does not collect, store, or transmit personal data, gameplay prompts, save data, or AI-generated dialogue. When the game is purchased through a store platform, the developer may receive purchaser/order information made available by that platform, as described below.

What the Game Itself Does Not Collect

  • Personal information entered or generated inside the game
  • Gameplay data, save data, or play history
  • Usage statistics or telemetry
  • Prompts entered by the player or AI-generated responses
  • Crash reports (no automatic submission)

Local Data Storage

The game stores the following data only on the user's PC, never on external servers:

  • Game settings such as language and sound
  • Save data such as current run progress, achievements, and character unlocks
  • AI model files bundled with the game, stored in the installation directory
  • Optional recovery-downloaded AI model files, stored in the user's userData directory, if the bundled model is missing

Settings, save data, and any recovery-downloaded model files can be deleted by uninstalling the game and removing the userData directory. The bundled model file is removed with the game installation.

AI Model and Local Inference

This game uses a locally-installed Large Language Model (LLM) for AI character behavior. Official builds include the model file with the game. If the bundled model is missing, the game may offer a one-time recovery download from a public model repository such as Hugging Face. During that recovery download, Hugging Face or network providers may receive ordinary network metadata such as IP address and request information under their own privacy policies.

  • All AI inference runs on the user's hardware.
  • No prompts, dialogues, or other game data are sent to external servers during gameplay.
  • No internet connection is required for normal gameplay.

This game is distributed via Steam, operated by Valve Corporation. Steam may collect data such as purchase history, playtime, and achievements according to Steam's own Privacy Policy. Please refer to Steam's Privacy Policy.

The game developer has no access to or control over data collected by Steam.

Other Store Platforms

If this game is purchased or downloaded through itch.io or BOOTH, those platforms may collect account, purchase, download, payment, or access-log data according to their own privacy policies. Depending on the platform's seller tools, the developer may be able to view purchaser or order information such as name, email address, country/region, payment/order IDs, or access logs. The developer does not receive gameplay prompts, AI-generated dialogue, or local save data from those platforms.

Third-Party Services

This game does not use any third-party analytics, advertising, or tracking services.

Children's Privacy

This game is not directed at children under 13. The game itself does not collect in-game data from any users, including children.

Changes to This Policy

This privacy policy may be updated to reflect changes in the game or applicable laws. The latest version will be available at the URL specified on the Steam store page.
